# MEDZU

## About MEDZU

MEDZU es una empresa orientada a la distribución de productos ferreteros, especializada en pernos, herramientas y suministros industriales. Su propósito es ofrecer soluciones eficientes, confiables y de alta calidad tanto para profesionales como para empresas del sector construcción e industria.

En un entorno donde el tiempo y la precisión son factores críticos, MEDZU busca optimizar el proceso de adquisición de productos mediante una plataforma que permite identificar necesidades específicas, recomendar productos adecuados y garantizar una entrega rápida, incluso el mismo día.

## Summary

## Features

- Centralización de productos ferreteros en una sola plataforma  
- Servicio de entrega el mismo día  
- Disponibilidad de ofertas en herramientas seleccionadas  
- Comercialización de productos de marcas reconocidas  

---

## How MEDZU Works

### 1. Dinos qué necesitas y tus preferencias

El usuario especifica los productos requeridos, como pernos o herramientas, así como sus preferencias. A partir de esta información, el sistema presenta opciones adecuadas según el tipo de proyecto.

### 2. Descubre ofertas y productos recomendados

La plataforma proporciona recomendaciones personalizadas y acceso a ofertas en herramientas seleccionadas. El usuario puede comparar, modificar o añadir productos según sus necesidades.

### 3. Recibe tus productos el mismo día

Una vez confirmado el pedido, este es procesado y entregado en el menor tiempo posible, ofreciendo flexibilidad en cuanto a horarios y dirección de entrega.
